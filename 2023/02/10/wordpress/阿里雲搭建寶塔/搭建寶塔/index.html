搭建宝塔

1.進入工作台，一般會提示你更改密碼，先更改然後在 root 鏈接

2.執行安裝命令，根據提示輸入y回車確認安裝即可（圖示系統版本為Centos 7.9）。
yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh

3.程式安裝完成後，提示資訊中「外網面板位址」、「內網面板位址」以及預設帳號」username“（8位隨機數位字母組合）與密碼”password“（8位隨機數位字母組合）即為寶塔面板頁面相關登錄資訊，保存好上述資訊用於步驟5使用瀏覽器登錄寶塔面板。


4.如果複製鏈接無法訪問到寶塔面板是需要開發寶塔端口 以8888 為例

5.有的要去安全組開發，有的在控制臺-安全-防火墻 設置也可以，我的就是在 控制臺-安全-防火墻 添加的

6.打開瀏覽器（建議使用Chrome瀏覽器）位址欄輸入步驟3獲取的“外網面板位址”對應位址（內部網络訪問輸入“內網面板位址”對應位址），回車後進入寶塔 Linux面板，“帳號”處輸入步驟3獲取的默認使用者名“username”對應的8位隨機數位字母組合、“密碼”處輸入步驟3獲取的密碼“password”對應的8位隨機數位字母組合，點擊” 登錄“即可。

7.登錄成功后，需要先綁定寶塔帳號。 如果暫未開通寶塔帳號，請註冊后登錄。
