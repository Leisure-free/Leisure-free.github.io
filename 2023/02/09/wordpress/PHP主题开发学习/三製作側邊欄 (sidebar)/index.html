三.製作側邊欄 sidebar.php

wordpress主題開發時，側邊欄代碼應寫在sidebar.php中。

  • 在主題目錄下創建檔sidebar.php。

  • 在文件內隨意輸入一段代碼

    <p>我是侧边栏内容</p>

  • 然後在首頁index.php中導入側邊欄檔，使用這個函數：

    <?php get_sidebar(); ?>

首先需要註冊側邊欄

要使用側邊欄功能，需要先註冊激活側邊欄。 在functions.php中添加如下代碼即可啟動：

方法一

// 自定义侧边栏
function my_custom_sidebar() {
    register_sidebar(
        array (
            'name' => '五弟主题的侧边栏',//侧边栏名称
            'id' => 'custom-side-bar',//侧边栏ID
            'description' => '这里是侧边栏的描述',//侧边栏描述
            'before_widget' => '<div class="widget-content">',//侧边栏前面的代码
            'after_widget' => "</div>",//侧边栏后面的代码
            'before_title' => '<h3 class="widget-title">',//侧边栏标题的前面的代码
            'after_title' => '</h3>',//侧边栏标题的后面的代码
        )
    );
}
add_action( 'widgets_init', 'my_custom_sidebar' );

add_action（） 這個函數，和字面意思一樣，是添加一個動作的意思。 第1個參數是添加到什麼地方，第2個參數是要添加的函數。

所以，上面這段代碼的意思就是在元件初始化時，執行 my_custom_sidebar（）;

不想這麼麻煩的話，也可以直接寫成這樣，都是可以註冊激活側邊欄的。
