认识几个常用的 php 函数

wp_head() ;

这是一个钩子函数，里面有许多内置的代码，很多插件要插入head部分代码时，都是通过这个钩子函数实现的，所以，大部分主题的head部分都包含这个函数

body_class()

这个函数根据不同的页面类型为body标签生成class选择器，从而让设计人员可以各方便灵活的控制不同页面中的各个元素

is_home()和is_front_page()

一般情况下，判断是不是首页是使用is_home()这个函数。但有些情况下，这个函数并不适用。比如你的首页并不是是使用的index.php，而是在后台指定了一个单独页面作为首页，这时用is_home()返回的就是假。

is_front_page()这个函数，不管你使用默认首页，还是独立页面作为首页。都会返回真。

所以，我这里希望是仅设计默认主页，就这么写：

if ( is_front_page() && is_home() ) {}

bloginfo()

这个函数用来获取你的博客网站的一些基本信息的，比如 bloginfo(“name”) 获取博客名称。

home_url()

这个函数用来获取你的网站首页的链接。比如我的网站获取到的就是：https://wplearning.org 。给其添加一个参数，home_url( ‘/’ ) ，这时获取的网站链接是https://wplearning.org/ ，在链接的最后面多了一个斜杠

esc_url()

此函数主要用于 URL 过滤，它会消除无效字符和删除危险字符。将字符转换成 HTML 实体,并且将 & 和 单引号(’) 转换成数字实体:& 。这个函数了解以下就行。在使用到URL的地方，调用一下它就可以了。
